# SOP

`SOP` 是一个给学生使用的设备操作助手 iPhone App。学生扫描设备上的 QR code 后，可以下载对应的 PDF、文本、Word 或视频资料，并按设备类型保存到本机。

当前包含：

- 实时相机扫码界面
- QR code 扫描结果识别
- PDF / Text / Word / 视频文件下载到本机
- 按设备类型分类保存 SOP 资料
- 本地资料库与文件预览
- 相机权限处理
- 可直接在 Xcode 中打开的 `SOP.xcodeproj`

打开方式：

1. 用 Xcode 打开 `SOP/SOP.xcodeproj`
2. 选择 iPhone Simulator
3. 配置自己的 Signing Team 后运行
