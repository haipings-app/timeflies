# Time Left

Time Left 是一款运行在 macOS 上的时间紧迫感管理软件，用来帮助用户直观看到“某件事还剩多少时间”，并围绕截止时间拆解任务、追踪完成进度、接收定期提醒。

当前仓库处于设计阶段，待设计确认后再进入实现。

## 目标问题

很多任务之所以被拖延，不是因为没有日历，而是因为：

- 看不到“剩余时间”带来的紧迫感
- 不能把一个时间目标拆成可执行事项
- 缺少持续、主动的提醒机制
- 完成过程没有被记录和反馈

Time Left 的核心目标是把“截止时间”变成一个持续可见、可执行、可提醒、可追踪的系统。

## 核心功能

1. 创建一个时间目标
   - 输入名称
   - 输入开始时间、结束时间
   - 或直接输入持续时长/时间间隔
2. 为目标添加具体事项列表
   - 支持逐项完成
   - 支持备注
3. 定期提醒
   - macOS 本地通知
   - Email 提醒
   - SMS 提醒（通过第三方服务）
4. 勾选完成事项
   - 支持完成状态
   - 支持完成时间和注释
5. 运行环境
   - 原生 macOS 应用

## 当前实现状态

仓库已经包含一个第一版 `SwiftUI` 原生骨架，覆盖：

- 时间目标列表与详情视图
- 开始/结束时间或时间间隔创建目标
- 事项新增、勾选、备注、删除
- 本地 JSON 持久化
- macOS 本地通知调度雏形
- Email / SMS 提醒配置入口
- 可直接在 Xcode 中打开的 `TimeLeft.xcodeproj`

说明：

- 当前机器的 Command Line Tools 与 SDK 版本不匹配，`swift build` 暂时无法在这里完成验证。
- 要生成标准 macOS `.app` 并继续界面开发，建议安装完整 Xcode 并切换到对应开发者目录。

## 当前结论

设计阶段建议优先采用：

- `SwiftUI + SwiftData` 构建原生 macOS 应用
- `UserNotifications` 做本地提醒
- Email 作为第一批外部提醒渠道
- SMS 通过可选第三方服务集成，不作为第一版阻塞项
